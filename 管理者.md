# 管理者功能說明文件

## 概覽

本系統提供管理者專用頁面，用於查詢捐血民眾資料並更新禮品領取狀態。

---

## 頁面清單

| 頁面 | 網址 | 說明 |
|------|------|------|
| 管理者登入 | `admin-login.html` | 輸入密碼驗證身分 |
| 禮品領取管理 | `admin-gift.html` | 查詢資料 / 更新禮品領取狀態 |

> 兩個頁面**不放在首頁入口**，需直接用網址進入。

---

## 登入機制

### 密碼儲存方式

- 密碼以 **SHA-256 hash** 儲存
- Hash 值不會出現在任何前端程式碼或 GitHub 儲存庫中

### Token 驗證機制

1. 管理者輸入密碼 → 前端送出至 GAS
2. GAS 將輸入密碼 SHA-256 後與儲存的 hash 比對
3. 比對成功 → GAS 產生一個**小時級 token**（`SHA-256(hash:當前小時數)`）回傳
4. Token 存於瀏覽器 `sessionStorage`（關閉分頁即自動清除）
5. 後續每次 API 呼叫都帶上 token，GAS 驗證後才執行操作
6. Token 有效期：**1 ~ 2 小時**（跨整點時有一小時緩衝）

### Session 失效情境

| 情境 | 結果 |
|------|------|
| 關閉瀏覽器分頁 | sessionStorage 清除，需重新登入 |
| Token 逾時（超過 2 小時） | API 回傳 `unauthorized`，自動跳轉登入頁 |
| 手動點擊「登出」 | 清除 sessionStorage，跳轉登入頁 |

## 初次設定密碼

在 Google Apps Script 編輯器中，貼上以下函式並執行一次，執行後刪除：

```javascript
function setAdminPassword() {
  const plain = '這裡填你的密碼';
  const digest = Utilities.computeDigest(Utilities.DigestAlgorithm.SHA_256, plain);
  const hash = digest.map(function(b) { return ('0' + (b & 0xff).toString(16)).slice(-2); }).join('');
  PropertiesService.getScriptProperties().setProperty('ADMIN_PASSWORD', hash);
  Logger.log('已儲存 hash：' + hash);
}
```

執行步驟：
1. 開啟 Google Sheet → 擴充功能 → Apps Script
2. 在編輯器底部貼上上方函式
3. 函式選單切換為 `setAdminPassword` → 點擊 ▶ 執行
4. 確認執行記錄顯示 hash 值後，刪除此函式
5. 儲存並**重新部署** GAS（管理部署作業 → 新版本 → 部署）

> 之後若要變更密碼，重複以上步驟即可，**不需修改前端程式碼**。

## GAS 新增函式說明

| 函式 | Action 名稱 | 說明 |
|------|------------|------|
| `adminLogin` | `admin_login` | 驗證密碼，回傳 token |
| `validateAdminToken` | — | 內部驗證 token（不對外暴露） |
| `adminLookup` | `admin_lookup` | 依手機電話或報到編號查詢資料 |
| `updateGift` | `update_gift` | 更新指定報到編號的禮品領取欄位（單欄位更新） |
| `adminUpdate` | `admin_update` | 以電話定位，全欄位更新（A～J 欄，電話不可修改） |

## admin-gift.html 操作流程

```
1. 開啟頁面 → 自動檢查 sessionStorage token
   └─ 無 token → 跳轉 admin-login.html

2. 選擇查詢方式（Tab 切換）
   ├─ 手機電話：輸入 09xxxxxxxx
   └─ 報到編號：輸入 4 位數字

3. 查詢結果預填至編輯表單（10 個欄位）
   ┌─────────┬──────────────────────────┐
   │ 報名時間 │ yyyy/mm/dd               │
   │ 公司行號 │ 下拉選單（含「其他」自填）  │
   │ 姓名     │ 文字輸入                  │
   │ 電話     │ 唯讀，不可修改             │
   │ 捐血CC數 │ 下拉選單（250cc / 500cc） │
   │ 報到時間 │ yyyy/mm/dd               │
   │ 報到編號 │ 4 位數字                  │
   │ 報到     │ —（未報到）/ Y（已報到）  │
   │ 捐血成功 │ —（未填）/ Y / N         │
   │ 禮品領取 │ —（未填）/ Y / N / 無符合資格 │
   └─────────┴──────────────────────────┘

4. 修改任意欄位後按下「確認更新」
   → 呼叫 admin_update，更新 Google Sheet A～J 欄（電話除外）

5. 按下「返回查詢」→ 清空表單，可繼續查詢下一位
```

## Google Sheet 欄位對應

| 欄 | 欄位名稱 | 說明 |
|----|---------|------|
| A | 報名時間 | 報名日期 |
| B | 公司行號 | 來源單位 |
| C | 姓名 | |
| D | 電話 | 手機號碼（文字格式） |
| E | 捐血CC數 | 250cc / 500cc |
| F | 報到時間 | 報到日期 |
| G | 報到編號 | 4 位流水號（文字格式） |
| H | 報到 | Y / 空白 |
| I | 捐血成功 | Y / N |
| **J** | **禮品領取** | **Y / N / 無符合資格（捐血失敗時系統自動填入；管理者可手動修改）** |
